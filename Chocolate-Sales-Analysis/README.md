## Overview
This project analyzes chocolate sales data to identify revenue trends, profitability, and business performance.

## Key Insights
- Orange Choco Syrup is the top-performing product
- Revenue peaks during January and February, with a dip in April
- APAC region contributes highest revenue
- Delish team performs best, while Yummies team lags

## Tools Used
- Power BI
- SQL
- Excel

## Features
- Interactive dashboard
- Monthly trend analysis
- Product and regional performance
- Drill-down analysis with filters
